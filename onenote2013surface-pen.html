如何把OneNote2013设置为surface pen开启默认？

稍微麻烦一下下即可拯救你的 surface pen ， 单击双击功能都可以回归！
1. 下载 autohotkey
AutoHotkey: macro and automation Windows scripting language
选择“ExpressInstallation”，
选择“exit”
2.创建脚本
在资源管理器，随便选一个文件夹，单击鼠标右键，选择“新建AutoHotkey的脚本”。
右击文件, 在记事本中进行编辑，再加入：
#F20:: run, Send #+n
return
#F19::Send #+s
return
3.保存文件，然后双击它在资源管理器中运行。
现在两个功能都正常了（前提是你的 send to onenote 是启动的）
4.如果它的正常，唯一剩下的事情就是将它添加到你的开机启动：
右键这个脚本，然后选择“创建快捷方式”
用鼠标右键单击Windows按钮，选择“运行”，然后键入“shell:startup”，然后按OK - 这个应该打开你的启动文件夹。
把快捷方式拖到里面。
